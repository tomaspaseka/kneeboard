# Architecture

**Kneeboard** is a .NET MAUI pilot kneeboard viewer — a single-document app that opens `.kneeboard` JSON files and displays mission content as full-screen swipeable pages organized into tabs. Primary platform: Windows 11 tablet. Secondary: Android.
